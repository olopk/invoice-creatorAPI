# Invoice-CreatorAPI

## Used technologies

- NodeJS (ExpressJS)
- GraphQL
- SOAP
- JWT
- MongoDB (Mongoose)

## Description

Features:
- Authentication ( signup, signin )
- API Data validation
- CRUD for customers, products, invoices, receipts.
- Serving data as GraphQL API,
- Fetching data using SOAP

## About

Backend of private project for commercial usage.
